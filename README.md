# 📊 Executive Sales Intelligence Dashboard

A consulting-grade data analytics project that demonstrates how businesses can transform raw sales data into actionable insights through data engineering, exploratory analysis, machine learning, and interactive dashboards.

This project simulates a real client engagement for a retail company, showcasing the complete analytics lifecycle—from cleaning raw data to delivering executive-level dashboards and business recommendations.

---

## 🎯 Project Objective

Businesses often rely on spreadsheets and disconnected reports, making it difficult to answer questions such as:

- Which products generate the highest profit?
- Which customers contribute the most revenue?
- Which regions are underperforming?
- How do discounts impact profitability?
- What can we expect sales to look like over the next few months?

This project demonstrates how these questions can be answered using modern data analytics techniques.
